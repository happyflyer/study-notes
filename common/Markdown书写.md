## 1. 字体

```markdown
常规
**粗体**
_斜体_
**_加粗斜体_**
~~删除线~~
```

## 2. 脚注

```markdown
这是第一句[^1]。这是第二句[^2]。这是倒数第二句[^n]。这是最后一句[^n]。
[^1]: 第 1 条的脚注
[^2]: 第 2 条的脚注
[^n]: 第 n 条的脚注
```

## 3. 转义

| 转义字符 | 书写       | 表示     |
| -------- | ---------- | -------- |
| 空格     | `&nbsp;`   | &nbsp;   |
| 小于号   | `&lt;`     | &lt;     |
| 大于号   | `&gt;`     | &gt;     |
| 和号     | `&amp;`    | &amp;    |
| 双引号   | `&quot;`   | &quot;   |
| 单引号   | `&apos;`   | &apos;   |
| 版权     | `&copy;`   | &copy;   |
| 注册商标 | `&reg;`    | &reg;    |
| 乘号     | `&times;`  | &times;  |
| 除号     | `&divide;` | &divide; |
